Printing a Java Applet

The print command in combination with Java applets does not work in all situations.

If you have "Print preview" (e.g. in Safari after issuing "Print"), check if the result is as expected - there still may be differences like missing vertical time line.
The safest way to print what you see is taking a screenshot.

To create a screenshot from a Java applet, follow these steps:

Mac OS X: Screenshot to clipboard

  • With the image you want to print displayed on the screen, press the key combination
          control+command+shift+4
  • After releasing the key combination, press the mouse and drag the light gray area across the screen.  When you release the mouse, the image is invisibly put onto the clipboard
  • Open e.g. an MS Word document and issue "Paste",
    - or go to application Preview and issue menu File >New from Clibpoard.
  • Now you can save or print
  • See also Apple menu> System Preferences> Keyboard&Mouse>Keyboard Shortcuts

Windows PC:

  • With the image you want to print displayed on the screen, press the "print screen" key on your keyboard.  (It is usually on the top row of the keyboard.)
  • Open any graphics program (like Paint), and press CTRL-V to paste in the image.  Crop the image to eliminate everything except the image area you want.
  • Print the image from the graphics program.
