PeakFinder

Author: Norbert Vischer (vischer at uva.nl)
History: 2013-Nov-09: First version
Source: PeakFinder.txt
Description: Detects peaks along a straight line, indicating peak positions as 'handles'
Features:
  • Live peak indication while dragging a line
  • Suppress neighbor peaks closer than minDistance
  • Adjustable dominance when suppressing neighbors
  • Add or delete peaks manually
  • Profile plot with peak markers
  • Numerical output
This is done by suppressing neighbor peaks where necessary. The user can set the dominance string to "A" (=highest amplitude), "L" (=Left) or "R" (=Right), and thus control which of the neighbors will survive. Terms 'left' and 'right' refer to the position in the profile plot. For example, if periodic objects appear with double contour borders, you could consistently pick only the first one by choosing dominance = "L". Macro Commands:
Create Demo Stack [1]

Creates a hyperstack to test perspective view of circular objects

Make Perspective [2]

Uses current (hyper)stack (or selection) to create perspective montage. You can double-click the eyedropper tool to adjust foreground color (used for border) and background color before issuing this command.

Save As PNG with Transparent Background [3]

Image is converted to 8-bit color, with the background declared as transparent. Programs like PowerPoint will honor transparency. For adding shadow as shown in Fig 2, choose in PowerPoint Format>Picture>Shadow

Save Animation as AVI [4]

Saves expanding stack as .avi movie. You may need to convert to other codecs when importing the movie elsewhere.
